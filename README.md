# Pomodoro Timer 🍅

「動き出し」をサポートする、新しいポモドーロタイマー。1分間のアイドリングで作業興奮を引き出し、スムーズに集中モードへ移行します。作業時間の記録・グラフ表示、PiP（ピクチャーインピクチャー）対応。

🌐 **[https://yuhi-sa.github.io/pomodoro/](https://yuhi-sa.github.io/pomodoro/)**

## ✨ 特徴

- 🆓 **完全無料・登録不要** - ブラウザだけで即利用
- 🔒 **プライバシー重視** - データはすべてブラウザ内（localStorage）に保存
- 📱 **レスポンシブ対応** - スマホ・PC対応
- 🖼️ **PiP対応** - ミニプレイヤーで他の作業をしながらタイマー確認
- 🔔 **通知音** - Web Audio APIによるビープ音（ON/OFF切替可能）
- 📊 **ダッシュボード** - 作業時間の記録・7日間の棒グラフ表示

## 💡 コンセプト

従来のポモドーロテクニック（25分作業 + 5分休憩の繰り返し）では、定期的な休憩のたびに「動き出し」のコストが発生します。本アプリは3つの思想でこの問題を解決します。

1. **定期的な休憩を排除** - アプリ側で休憩タイミングを強制しない。休憩はユーザーが自分で決める
2. **「動き出し」をサポート** - 1分間のアイドリングで作業興奮を引き出し、スムーズに作業へ移行
3. **報酬系を刺激** - ダッシュボードで作業実績を可視化し、達成感を得られる

## ⏱️ タイマーの流れ

```
スタート → アイドリング（1分） → 選択 → 作業（無制限） → 休憩（無制限） → アイドリングへ戻る
                                   ↓
                              1分休憩 → アイドリングへ戻る
```

| フェーズ | 時間 | 説明 |
|----------|------|------|
| アイドリング | 1分 | 「1分だけ」の超短期作業でウォームアップ |
| 自動移行 | 5秒 | アイドリング後、操作がなければ自動で作業へ |
| 作業 | 無制限 | ユーザーが休憩を選ぶまで集中を継続 |
| 小休憩 | 1分 | アイドリング後に選択した場合の短い休憩 |
| 休憩 | 無制限 | ユーザーが終了を選ぶまで自由に休憩 |

## 🧩 機能一覧

| 機能 | 説明 |
|------|------|
| 円形タイマー | SVGプログレスリングでアイドリング/小休憩の進捗を視覚化 |
| 通知音 | Web Audio APIによるビープ音。設定でON/OFF可能 |
| 自動移行 | アイドリング完了後5秒で自動的に作業モードへ。設定でON/OFF可能 |
| ダッシュボード | 今日の作業時間・セッション数・累計時間・過去7日間の棒グラフ・セッション履歴 |
| PiP（ピクチャーインピクチャー） | ミニプレイヤーで他のタブを開きながらタイマーを確認・操作 |
| データ永続化 | localStorageに設定・作業履歴を保存。ページを閉じてもデータが残る |
| バックグラウンド対応 | Date.now()ベースのタイマーにより、タブ非表示時もずれなく動作 |

## 🚀 ローカル開発

```bash
git clone https://github.com/yuhi-sa/pomodoro.git
cd pomodoro
# 任意のHTTPサーバーで起動
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` を開きます。
